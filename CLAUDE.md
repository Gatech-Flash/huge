## Project Overview

**huge** (High-Dimensional Undirected Graph Estimation) is a dual R+Python package for sparse graphical model estimation and inference in high dimensions. The core algorithms are implemented in C++ using direct BLAS calls for linear algebra and OpenMP for parallelism (RcppEigen dependency removed in 1.5).

Two package variants share this repo:
- **R package (`huge`)**: CRAN-published, lives at the repo root
- **Python package (`pyhuge`)**: native Python (no R runtime needed), lives under `python-package/`

## Build & Test Commands

### R package

```bash
# Build the package (from repo root)
R CMD build .

# Check (CRAN-style, runs tests + examples)
R CMD check huge_*.tar.gz

# Install locally
R CMD INSTALL .

# Run tests only
Rscript tests/test.R
```

The `configure` / `configure.ac` system detects OpenMP support and generates `src/Makevars` from `src/Makevars.in`. On macOS you need `brew install llvm` and `~/.R/Makevars` pointing to clang-omp.

### Python package (`pyhuge`)

```bash
cd python-package

# Install editable with all dev deps
pip install -e ".[dev]"

# Run tests
pytest

# Build native C++ extension (pybind11) — happens automatically via setup.py
pip install -e .

# Build docs locally
mkdocs build --strict

# Build distribution
bash scripts/build_dist.sh

# Bump version + release
python scripts/bump_version.py 0.8.x
bash scripts/release.sh 0.8.x

# Diagnostics
pyhuge-doctor
python -c "import pyhuge; print(pyhuge.test())"
```

## Architecture

### R package (repo root)

- `R/` — R source files, one per exported function:
  - `huge.R` — main dispatcher, calls method-specific functions
  - `huge.glasso.R` — graphical lasso estimation
  - `huge.mb.R` — Meinshausen-Buhlmann neighborhood selection
  - `huge.tiger.R` — TIGER (tuning-insensitive graph estimation)
  - `huge.ct.R` — correlation thresholding
  - `huge.select.R` — model selection (StARS, RIC, eBIC)
  - `huge.npn.R` — nonparanormal transform (preprocessing)
  - `huge.generator.R` — synthetic graph/data generation
  - `huge.inference.R` — statistical inference on graph edges
  - `huge.roc.R` — ROC curve computation
  - `huge.plot.R` — visualization
  - `RcppExports.R` — auto-generated Rcpp bindings
- `src/` — C++ backend (compiled to `huge.so`):
  - `hugeglasso.cpp` — graphical lasso solver (coordinate descent)
  - `SPMBgraph.cpp` / `SPMBgraphsqrt.cpp` — MB graph estimation kernels
  - `RIC.cpp` — rotation information criterion
  - `SFGen.cpp` — scale-free graph generator
  - `RcppExports.cpp` — auto-generated
  - `Makevars.in` → `Makevars` — OpenMP flags via configure
- `NAMESPACE` — auto-generated by roxygen2; exports all `huge.*` functions
- `man/` — roxygen2-generated Rd documentation

### Python package (`python-package/`)

- `pyhuge/core.py` — all estimators, selection, transforms, plotting (pure Python + numpy/scipy)
- `pyhuge/__init__.py` — public API re-exports
- `pyhuge/doctor.py` — `pyhuge-doctor` CLI diagnostic tool
- `pyhuge/parity.py` — R-vs-Python parity checking utilities
- `pyhuge/data/` — bundled datasets (`stockdata.npz`)
- `cpp/` — pybind11 native acceleration kernels (`_native_core`):
  - `native_core_bindings.cpp` — Python bindings
  - `native_core_kernels.cpp` — glasso/mb/tiger C++ kernels
  - `native_generator.cpp` — generator C++ kernel
  - `native_select.cpp` — selection C++ kernel
- `tests/` — pytest suite (unit, e2e, native smoke, R parity tests)
- `docs/` — MkDocs documentation source
- `scripts/` — release and docs helper scripts

### Shared pipeline concept

Both packages implement the same modeling pipeline:
1. **Generate** synthetic data (`huge.generator` / `huge_generator`)
2. **Preprocess** via nonparanormal transform (`huge.npn` / `huge_npn`)
3. **Estimate** graph structure (`huge` with method = `glasso`/`mb`/`tiger`/`ct`)
4. **Select** regularization parameter (`huge.select` / `huge_select` with criterion = `stars`/`ric`/`ebic`)
5. **Infer** edge significance (`huge.inference` / `huge_inference`)
6. **Evaluate** via ROC (`huge.roc` / `huge_roc`)
